# .dotfilesMinimal
My setups used for my raspberry pi's/ VM's etc  

## __INSTALLING__  
clone the repo, then
* To install both vim and bashrc configs:
````
make
````
* To just intall vim configs
````
make vim
````
* To just install bashrc configs
````
make bash
````
##__TL;DR__  
just run this  
`cd;  
mkdir repos;  
cd repos;  
git clone https://github.com/skippy404/.dotfilesMinimal;  
cd .dotfilesMinimal;  
make
`
